# Creepy Jokes

Creepy Jokes for your next PHP project.

## Installation

Require the package using composer.

```bash
composer require amasir20/creepy-jokes
```

## Usage

```php
use Amasir20\CreepyJokes\JokeFactory;

$jokes = new JokeFactory();
$joke = $jokes->getRandomJoke();

```
